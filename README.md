# JavaJukeBox-2.1
	
HOW TO COMPILE AND RUN:	

1.) Unzip the folder
2.) open a terminal
3.) cd inside of the folder of the project where the pom.xml is located
4.) execute the command: mvn package
5.) cd to ./target
6.) execute the command:  java -jar ./JavaJukeBox-2.0-1.0-SNAPSHOT.jar
7.) listen to music
